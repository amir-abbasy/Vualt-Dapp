```shell
 # run Node app Vault Contact  Deposit & Withdraw Event Listener

```

## Manual execution
#### Harhat contract test
```shell
cd hardhat
npm i

npx hardhat compile
npx hardhat test
```

#### Run Contract Event Listener Service
```shell
cd hardhat
npm i

npm start
```


#### Frontend Dapp run
```shell
cd frontend
npm i

npm run dev
```
